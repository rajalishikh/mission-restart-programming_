1. What is the difference between null and undefined?
undefined: যখন আমি  একটি ভেরিয়েবল ডিক্লেয়ার করি কিন্তু তাতে কোনো ভ্যালু অ্যাসাইন করি  না, তখন জাভাস্ক্রিপ্ট ডিফল্টভাবে সেটিকে undefined হিসেবে ধরে নেয়। এটি একটি টাইপ।
null: এটি একটি অ্যাসাইনমেন্ট ভ্যালু। যখন ডেভেলপার ইচ্ছা করে কোনো ভেরিয়েবলকে খালি বা "শূন্য" বোঝাতে চান, তখন null ব্যবহার করা হয়।

2) What is the use of the map() function in JavaScript? How is it different from forEach()?
map(): এটি একটি অ্যারের প্রতিটি উপাদানের ওপর কাজ করে   মানে  অ্যারের  সব গুলা উপদান এর উপর একটা একটা করে যাই এবং ফলাফল হিসেবে একটি নতুন অ্যারে রিটার্ন করে। এটি অরিজিনাল অ্যারেকে পরিবর্তন করে না।
forEach(): এটি অ্যারের প্রতিটি উপাদানের জন্য একটি ফাংশন কল  মানে সব গুলা উপর একটা একটা করে যাই করে, কিন্তু এটি কোনো কিছু রিটার্ন করে না। এটি মূলত সাইড ইফেক্ট (যেমন: কনসোলে প্রিন্ট করা) এর জন্য ব্যবহৃত হয়। 

3) What is the difference between == and ===?
== (Loose Equality): এটি শুধুমাত্র ভ্যালু বা মান চেক করে। যদি ডাটা টাইপ আলাদা হয়, তবে এটি টাইপ কনভার্সন (Type Coercion) করে মান মিলিয়ে দেখার চেষ্টা করে। উদাহরণস্বরূপ: 5 == "5" সত্য (true) রিটার্ন করবে।
=== (Strict Equality): এটি মান এবং ডাটা টাইপ—উভয়ই চেক করে। এখানে কোনো টাইপ কনভার্সন হয় না। তাই 5 === "5" মিথ্যা (false) রিটার্ন করবে।




4) What is the significance of async/await in fetching API data?
জাভাস্ক্রিপ্ট সাধারণত অ্যাসিনক্রোনাসভাবে কাজ করে। async/await কোডকে এমনভাবে লিখতে সাহায্য করে যাতে তা দেখতে সিনক্রোনাস বা সাধারণ কোডের মতো মনে হয়। সহজ করে যদি বলি এতা সিরিয়াল  অনুযায়ী চলে 
এটি প্রমিস (Promise) হ্যান্ডেল করা সহজ করে দেয়।
.then() এবং .catch()-এর চেইনিং কমিয়ে কোডকে আরও পরিষ্কার এবং পড়ার যোগ্য (readable) করে তোলে।
try-catch ব্লক ব্যবহার করে খুব সহজে এরর হ্যান্ডলিং করা যায়।

5) Explain the concept of Scope in JavaScript (Global, Function, Block).
স্কোপ নির্ধারণ করে যে আপনার কোডের কোন অংশ থেকে কোন ভেরিয়েবল অ্যাক্সেস করা যাবে।
Global Scope: যদি কোনো ভেরিয়েবল যেকোনো ফাংশন বা ব্লকের বাইরে ডিক্লেয়ার করা হয়, তবে সেটি গ্লোবাল স্কোপে থাকে এবং কোডের যেকোনো জায়গা থেকে তা ব্যবহার করা যায়।
Function Scope: একটি ফাংশনের ভেতরে ডিক্লেয়ার করা ভেরিয়েবল শুধুমাত্র সেই ফাংশনের ভেতরেই কাজ করে। এর বাইরে থেকে একে অ্যাক্সেস করা যায় না।
Block Scope: let এবং const দিয়ে ডিক্লেয়ার করা ভেরিয়েবলগুলো শুধুমাত্র একটি নির্দিষ্ট সেকেন্ড ব্র্যাকেট { ... } বা ব্লকের ভেতরে কাজ করে (যেমন: if স্টেটমেন্ট বা for লুপের ভেতর)।
